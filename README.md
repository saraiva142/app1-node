Para fazer o merge:

Supondo que: Estou numa branch, quero comitar o que eu fiz e depois fazer o merge com o main

Passos:
* git add .
* git commit -m "Commit..."
* git push origin <branch>

* git checkout main
atualizar a main:
* git pull origin main 
* git merge <branch-que-queira-fazer-o-merge>
* git push origin main
